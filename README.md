# AltairaLabs Helm Charts

Helm chart repository for [AltairaLabs](https://www.altairalabs.ai) products.

## Usage

```bash
helm repo add omnia https://charts.altairalabs.ai
helm repo update
helm search repo omnia
```

## Available Charts

Charts will be published here as they are released. See the [Omnia documentation](https://omnia.altairalabs.ai/tutorials/getting-started/) to get started.
